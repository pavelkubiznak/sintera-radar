Jak přidat ANTHROPIC_API_KEY

  1. Vytvoř API key na console.anthropic.com/settings/keys (přihlas se svým Anthropic účtem, sekce "API Keys" → "Create Key")
  2. Na VPS otevři {{ cfg.vps_path }} v editoru: 
    ssh root@178.104.207.97
nano {{ cfg.vps_path }}
  3. Najdi řádek (zakomentovaný) a odkomentuj + vlož hodnotu: 
    ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxx
  4. Restart služby: 
    systemctl restart sintera-radar
  5. Refresh této stránky — uvidíš zelený "✔ nastaveno". Příští enrichment (manuální nebo nightly cron) automaticky využije LLM vrstvu.
Náklady: Claude Haiku 4.5 ~$0.02 za firmu, takže 30 firem denně přes nightly cron = $0.60/den = ~$18/měsíc. Při lepší kvalitě extrakce typicky zlepšení o 10-15 % named contacts.

Co LLM extraction dělá navíc

  • Strukturovaná extrakce — Claude prochází plné textové verze kontaktních stránek a vytahuje kontakty jako JSON (jméno, role, email, telefon) s confidence skóre.
  • Lepší disambiguace — když je na stránce 5 emailů a 3 jména, LLM zvládne přesněji určit které email patří ke kterému jménu (vs současný text-proximity regex, který občas chybuje).
  • Detekce HR-specifických rolí — Claude rozumí, že "vedoucí náboru" / "talent acquisition lead" / "personální ředitelka" jsou HR. Regex katalog je omezený.
  • Tolerance ke JS-renderingu — když stránka obsahuje text ale ne ideálně formátovaný, LLM si poradí.
Bez API key Radar dál funguje normálně přes regex extrakci (současný stav). LLM je vrstva nahoru, ne náhrada.
