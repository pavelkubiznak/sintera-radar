Jak propojit s Google Looker Studio

1

Stahni CSV

Klikni na tlacitko "Stahnout CSV export" vyse. Soubor sintera_analytics.csv se ulozi na disk.

2

Nahraj do Google Sheets

Otevri Google Sheets, vytvor novy sesit, a importuj CSV (Soubor → Import → Nahrat).
Nebo pouzij Google Apps Script pro automaticky import — viz nize.

3

Pripoj Looker Studio

Otevri Looker Studio, pridej novy zdroj dat → Google Sheets → vyber sesit z kroku 2.
Sloupce je_aktualizovano, je_opakovane atd. pouzij jako filtry.

4

Automaticky import (volitelne)

Pro automaticky import po kazdem scanu pridej tento Apps Script do Google Sheets:

function importSinteraCSV() {
  var url = "http://178.104.207.97:5001/api/analytics.csv";
  var response = UrlFetchApp.fetch(url, {
    headers: {
      "Authorization": "Basic " + Utilities.base64Encode("sintera:radar2026")
    }
  });
  var csv = Utilities.parseCsv(response.getContentText());
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Data");
  if (!sheet) sheet = SpreadsheetApp.getActiveSpreadsheet().insertSheet("Data");
  sheet.clear();
  sheet.getRange(1, 1, csv.length, csv[0].length).setValues(csv);
}
// Nastav trigger: Edit > Triggers > importSinteraCSV > Time-driven > Daily
